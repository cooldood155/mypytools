# Supported

This document is here to explain all of the requirements this library supports. For example, this library requires a Python interpreter, so the supported Python interpreter versions are listed here. This includes dependency versions and anything else that is needed to be known.

## Python

- ### Version *(python3)*

  All Python interpreters with a version greater than or equal to `3.12` are compatible with this library.

## Dependencies

There are currently no dependencies for this library.
